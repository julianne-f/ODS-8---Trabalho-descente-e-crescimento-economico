# ODS 8 – Trabalho Decente e Crescimento Econômico

## Objetivo
Este projeto tem como objetivo analisar problemas relacionados ao trabalho e propor soluções baseadas no ODS 8.

## Sobre o Projeto
O projeto consiste na ideia de um aplicativo que funciona como um mapa de oportunidades, mostrando vagas próximas ao usuário e facilitando o acesso ao emprego.
